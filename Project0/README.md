# Hotel Booking App

The hotel booking app is a handy tool that links available rooms with users.

Technologies

	•	Visual Studio Code - 2019

	•	.Net Core - 5

	•	Azure Data Studio - 1.35.1

Features

	•	Login

	•	Add users

	•	Create/Update/Cancel a reservation

	•	Search for available rooms by state

To be Added
